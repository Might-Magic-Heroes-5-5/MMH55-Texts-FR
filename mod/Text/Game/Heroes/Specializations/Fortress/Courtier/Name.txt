Classe : Brigand<br>Spécialisation : Langue d'Or<br><br><body_bright>Augmente les chances pour que les créatures neutres rejoignent l'armée du Héros. Le héros bénéficie d'un bonus de +5% par niveau de puissance militaire pour les calculs diplomatiques. De plus, le Héros reçoit de l'Or et des ressources en cas de victoire militaire.<br><color_default>La quantité de chaque type de ressource gagnée après une victoire est aléatoire entre 0 et une valeur maximale ; Max(Or) = Coût*0,015, Max(Bois & Minerai) = Coût*0,0001, Max(Precious) = Coût*0,00005, où Coût est le coût d'embauche (en Or) des ennemis tués.
